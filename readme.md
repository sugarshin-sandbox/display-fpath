# display-fname

Display file name

```zsh
node app.js 'directory/path/to'

# directory/path/to/file1.txt
# directory/path/to/file2.txt
# directory/path/to/dir/file.txt
# directory/path/to/dir/dir2/file.txt
```

write file

```zsh
node app.js 'directory/path/to' filename.txt
```
